Student Performance Analyzer

A powerful data-driven tool designed to evaluate and visualize student performance using advanced analytics. This project provides insightful graphs and reports to help educators and students better understand academic progress.

📊 Class Performance Analysis

  • Collects and analyzes the marks of all students in a class
  • Generates comparative statistical graphs to highlight trends and patterns
  • Identifies top-performing and underperforming students
  • Helps educators understand subject-wise strengths and weaknesses

👤 Individual Student Analysis

  • Tracks and evaluates individual student's academic progress
  • Provides detailed performance trends through various charts
  • Allows identification of areas needing improvement
  • Generates personalized feedback and performance reports

📅 Attendance Analytics

  • Monitors attendance records throughout the semester
  • Displays attendance trends with interactive graphs
  • Identifies students with low attendance for intervention
  • Integrates with academic performance data

🎓 SGPA-Based Insights

  • Calculates and visualizes SGPA trends over time
  • Provides graphical representation of academic progression
  • Helps track semester-wise performance
  • Enables predictive analysis for future outcomes

Python Pandas Matplotlib NumPy

College Fest Management Website

Project Overview: Developed a dynamic and responsive website for college fest event management, registrations, and information sharing. The platform provides an intuitive user experience, allowing students to register for events, view schedules, and get real-time updates.

  • ✅ User Authentication: Secure login and registration for participants and organizers.
  • ✅ Event Registration: Automated fee calculation for solo and group events.
  • ✅ Real-Time Updates: Dynamic announcements and schedules.
  • ✅ Admin Dashboard: Manage event entries, registrations, and participant details.
  • ✅ Responsive & Fast UI: Tailwind CSS for a seamless, modern user experience.

Impact & Outcome:
🔹 Streamlined event registration, reducing manual paperwork.
🔹 Improved accessibility with a mobile-friendly design.
🔹 Enabled efficient data management for organizers.
🔹 Successfully hosted hundreds of participants.
